# 更新日志

## v0.0.8 (2025-01-24)

### 新功能
- **API 提供商扩展**: 新增 OpenRouter、Vercel AI 和 Ollama 本地模型支持
- **虚拟滚动**: 会话列表支持虚拟滚动，可流畅处理大量会话
- **自定义模型**: 支持添加自定义 API 模型配置
- **标签系统**: 新增会话标签、视图和颜色标记功能

### 改进
- 大幅优化代码架构，提升应用稳定性
- 修复多处类型错误，增强代码可靠性

---
